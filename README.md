# salary-calculator

# 概要
塾講師のアルバイトとして勤務する中で、給与計算を効率化するために開発したWebアプリです。
特定の塾システムの勤務データを取得し、給与を自動計算します。

＊公開版では塾名やログインURLなどの設定情報を除外しています。

# 使用技術
- Python
- Flask
- Playwright
- HTML/CSS

# 主な機能
- ログイン情報の入力
- 勤務データの自動取得
- 給与の自動計算
- 計算結果の表示

# 開発内容・工夫した点
実際に塾講師として勤務する中で、給与額を確認できるのが給与明細発効後のみであり、現在組まれているシフトから将来の給与額を把握できないことに不便を感じたため開発しました。

Playwrightを利用して勤務データを自動取得し、授業数や勤務情報をもとに給与を自動計算する機能を実装した。これにより、給与明細を待たずに現時点での想定給与を確認できるようになりました。

また、Flaskを用いてWebアプリ化することで、PCやスマートフォンから手軽に利用できるようにしました。
